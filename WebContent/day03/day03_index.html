Day03 Index

1.post_test.html
post_test.html은 사용자가 입력하는 화면의 url
request_test.jsp는 사용자 입력을 전달받아 처리하는 url
post 방식으로 요청을 보내다.(데이터 저장을 위한 처리)
2.get_test.jsp
get_test.html는 3가지 기능이 있다. 1)사용자의 목록을 출력하는 화면
2)고객번호로 조회할 수 있는 요청 링크 3)form을 이용한 조회
get_result.jsp는 사용자 입력을 전달받아 처리하는 url
get 방식으로 요청을 보낸다. (데이터 조회를 위한 처리)
3.get_result.jsp
get_test.html에서 전달받은 url: 조회를 원하는 고객번호를 전달받아 데이터를 수정할 수 있는 화면을 구현
파라미터 값 custno가 없으면 실행시 오류가 난다.
데이터 수정을 처리하는 서버 url(db테이블 데이터 수정) : member_update.jsp
